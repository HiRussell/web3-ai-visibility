Methodology

  • 50 hand-crafted crypto questions across categories: recommendations ("best DEX on Solana"), comparisons ("Aave vs Compound"), tactical ("how to leverage long ETH"), risk ("is Ethena safe"), and persona-realistic queries.
  • Each question runs through 4 AI search products via a single OpenRouter key: Perplexity Sonar Pro, OpenAI GPT-4o (with web search), Grok 4.20 (with live X data), and Gemini 2.0 Flash (with Google Search grounding). These are stable, widely-deployed API tiers — close to what most users actually see. Bleeding-edge variants (gpt-5.5, gemini-3-flash-preview) are tracked as a TODO once wallet budget + provider stability support them.
  • Protocol mentions extracted with regex against the DefiLlama canonical protocol list (~7,400 entries), filtered to drop chain slugs and high-frequency English-word slugs (TVL < $1M skipped). See failures.md for what false positives we've caught.
  • Open source: github.com/HiRussell/web3-ai-visibility · weekly snapshots committed every Monday 04:00 UTC.
